Additor

a synth based on the Web Audio API

Live Version Source Code
  • JavaScript
  • Web Audio API
  • Canvas
  • RequireJS

Additor is a browser-based additive synthesizer app built from scratch with vanilla JavaScript. I built a library of audio components based on the Web Audio API, and UI widgets based on the HTML5 canvas. The synthesis component is split up into Attack and release envelopes are specified as 2d arrays. Module bundling is done with RequireJS.

Weathervane

weather forecast app based on the Apixu API

Live Version Source Code
  • Angular JS
  • Google Maps API
  • Apixu Weather API

Weathervane is a simple weather forecast app. The app lets the user specify the location in a variety of formats, and geocodes the input using the Google Maps API. The received latitude and longtitude coordinates are then sent to the Apixu Weather API to retrieve the forecast. The app was structured using Angular JS. A generative background animation was created using the HTML canvas.

JSLiveWidgets

a mini-library of canvas-based widgets

Source Code Documentation
  • JavaScript
  • Canvas

JSLiveWidgets is a mini-library of canvas-based widgets for browser-based audio applications. The library was loosely inspired by and modeled on the look of Ableton Live and Max for Live UI components. I have been using Max extensively to prototype multimedia tools in previous projects, and have grown accustomed to having these UI components available. I thought it would be useful to have a similar set of components that are basic, customizable, and pluggable, implemented in JS for use in browser-based audio applications (and potentially in other environments as well via WebKitGtk). The project is a work-in-progress, but currently the library has the following components: keyboard, dial, slider, volume meter, envelope graph, histogram, drop-down menu, draggable number box. The library is implemented as class modules that have support for AMD and CommonJS, or can be used as globals. UI interaction is supported for mouse as well as touch events.
